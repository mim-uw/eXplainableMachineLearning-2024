Possible reasons of differences in feature important rankings between methods:

  • SHAP values can capture more complex interactions between features compared to permutation-based feature importance.
  • Gain-based feature tends to be biased towards features with bigger cardinality.
  • Permutation-based feature importance is more sensitive to the choice of the evaluation metric.
