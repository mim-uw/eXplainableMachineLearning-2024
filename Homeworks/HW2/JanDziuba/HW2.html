Task 1¶

Demographic parity¶

a - red group
b - blue group

$P(\hat{Y} = 1 \mid A = a) = 50\%$
$P(\hat{Y} = 1 \mid A = b) = 65\%$
$P(\hat{Y} = 1 \mid A = a)/P(\hat{Y} = 1 \mid A = b) \approx 0.77 < 0.8$
Demographic parity ratio is less than 0.8, so by this metric classifier is not fair.
It is worth noting that classes are not balanced, so it's not a perfect metric for this problem

Equal opportunity¶

$P(\hat{Y} = 1 \mid A = a, Y = 1) = 50\%$
$P(\hat{Y} = 1 \mid A = b, Y = 1) = 60/80 = 75\%$
$P(\hat{Y} = 1 \mid A = a, Y = 1)/P(\hat{Y} = 1 \mid A = b, Y = 1) \approx 0.67$
By this metric also classifier is not fair.

Predictive Rate Parity¶

$P(Y = 1 \mid A = a, \hat{Y} = 1) = 50\%$
$P(Y = 1 \mid A = b, \hat{Y} = 1) = 60/65$
$P(Y = 1 \mid A = a, \hat{Y} = 0) = 50\%$
$P(Y = 1 \mid A = b, \hat{Y} = 0) = 20/35$
$P(Y = 1 \mid A = a, \hat{Y} = 1)/P(Y = 1 \mid A = b, \hat{Y} = 1) \approx 0.54$
$P(Y = 1 \mid A = a, \hat{Y} = 0)/P(Y = 1 \mid A = b, \hat{Y} = 0) \approx 0.88$
So Positive predicted value is different, but negative predicted value is similar. By this metric classifier is not fair, but this time it is favoring the red group.

Starred task¶

To improve faireness we could:
Increase the quality of red group classifier.
Decrease the quality of blue group classifier.
Shift the number of people to be enrolled from different groups.
We can do this in both ways, depending on which metric we want to optimize.

Task 2¶

I picked the adult income dataset.
I chose gender to be the protected attribute.
I compared random forest with xgboost and then xgboost with resample, reweight and roc_pivot bias mitigation methods.

XGBoost performs better

XGBoost also behaves better across all fairness metrics

Resampling decreases all performance metrics.
ROC pivot performs the best from all mitigations increasing precision and auc.

Resampling also performs worst when it comes to fairness. It significantly worse in EOR PPR and PER than the base model.
Again ROC pivot performs best improving all fairness metrics of the base model

Since there are multiple metrics for performance and fairness it's hard to find a correlation between model preformance and fairness
