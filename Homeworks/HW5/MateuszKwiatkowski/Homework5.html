Task 1¶

$g_{1}^{PD}(z) = E_{X_2}f(X_{1|=z}) = E_{X_2}(z + x_2)^2 = E_{X_2}z^2 + 2E_{X_2} zx_2 + E_{X_2}x_2^2 = z^2 + 0 + E_{X_2}x_2^2 = z^2 + \frac{1}{2}\int_{-1}^1 x^2 dx = z^2 + \frac{1}{3}$

$g_{1}^{MP}(z) = E_{X_2|x_1 = z}f(X_{1|=z}) = E_{X_2|x_1 = z}(z + x_2)^2 = E_{X_2|x_1 = z}z^2 + 2E_{X_2|x_1 = z} zx_2 + E_{X_2|x_1 = z}x_2^2 = z^2 + 2z^2 + z^2 = 4z^2$

$g_{1}^{AL}(z) = \int_{-1}^z [E_{X_2|x_1 = v}\frac{\partial f(x)}{\partial x_1}]dv = \int_{-1}^z [E_{X_2|x_1 = v}[2x_1 + 2x_2]]dv = \int_{-1}^z [2E_{X_2|x_1 = v}x_1 + 2E_{X_2|x_1 = v}x_2]dv = \int_{-1}^z 4v dv = 2z^2 + c$
